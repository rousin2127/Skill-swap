

Project Name: SkillSwap – A Local Skill Exchange Platform

Purpose

SkillSwap is designed to connect people within the same community to exchange knowledge and skills without monetary transactions. The goal is to build a collaborative environment where users can learn something new or teach what they know, fostering local connections and personal growth.

live URL: https://skillswap-firebase-auth-c71c8.web.app/

Key Features

Skill Listings: Users can create detailed listings for the skills they offer or want to learn.
User Authentication: Secure signup/login with  Firebase Authentication.
Responsive Design: Optimized for all devices — mobile, tablet, and desktop.
Profile Management: Upload profile pictures,
